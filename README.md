# ember-later-shim

ES6 accessible module for laterjs within your EmberJS applications.

## Installation

```
ember install ember-later-shim
```

## Usage

```js
import later from 'later';
```

See the [LaterJS page](http://bunkat.github.io/later/) for further documentation.
